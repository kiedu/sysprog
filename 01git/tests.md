### Що таке Git?

Інструмент для роботи з базами даних
Фреймворк для створення мобільних додатків
Розподілена система керування версіями
Мова програмування для веброзробки

### Що таке розподілені системи контролю версій?

Системи, які не підтримують роботу з віддаленими репозиторіями
Системи, які зберігають записи про всі зміни файлів локально у базі даних
Системи, що працюють за допомогою створення повної копії репозиторію
Системи, що використовують єдиний сервер для зберігання усіх версій файлів

### Що таке репозиторій?

Програма для керування версіями коду
Місце, де зберігаються тільки відеофайли
Місце, де зберігаються тільки програмні засоби
Місце, де зберігаються та підтримуються будь-які дані

### Яка команда використовується для виведення та зміни конфігурації для роботи Git?

git push
git pull
git config
git commit

### Яку функцію виконує файл .gitignore?

Виконує підрахунок рядків в коді репозиторію
Дозволяє відключити контроль для файлів розташованих у репозиторії
Зберігає всі коміти, які були зроблені в репозиторії
Вказує, які файли мають бути додані до репозиторію

### Що таке стан зміненого файлу?

Файл, який було видалено з репозиторію
Файл, який під контролем git і з яким було щось зроблено
Файл, який було додано до репозиторію, але не було закомічено
Файл, який фізично зберігається у сховищі git

### Яка команда використовується для виведення історії комітів з метаданими?

git status
git commit
git add
git log

### Яка команда використовується для додавання файлів під версійний контроль?

git init
git commit
git status
git add

### Яка команда використовується для перемикання на вказаний коміт в історії?

git merge
git clone
git branch
git checkout

### Що робить команда git add?

Переводить файли зі станів невідслідковуваного та зміненого у стан проіндексованого
Змінює ім'я файлу або переміщує файл
Видаляє файл з файлової системи та додає цю зміну до індексу
Додає всі зміни до індексу та виконує коміт

### Який режим команди "git reset" є найбезпечнішим?

--hard
Немає правильної відповіді
--mixed
--soft

### Яка команда відправляє всі зміни до <local_brunch> на віддалений репозиторій?

git clone <url>
git fetch <remote>
git push <remote> <local_brunch>
git remote add <name> <url>

### Що робить команда git branch <name>?

Перемикає покажчик HEAD на іншу гілку
Додає новий коміт до гілки
Видаляє гілку з репозиторію
Створює нову гілку та покажчик на поточний коміт

### Встановіть порядок команд для створення нової гілки та створення в ній коміта
git add .	
git commit -m "Commit message"	
git branch bugfix	
git switch bugfix

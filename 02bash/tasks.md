### ПР22. Bash
1. Підготувати веб-сторінку опрацювання теми Linux, з наступними елементами:
* Коротка інформація про ОС GNU/Linux;
* Перелік найбільш популярних дистрибутивів;
* Приклади команд роботи в Linux CLI (pwd, echo, ls, cd, touch, mkdir, cp, mv, rm, history);
* BASH-cкрипт, що імітує авторизацію в системі в текстовому режимі;
* Джерела інформації по темі.
2. Оформити веб-сторінку використовуючи розмітку та стилі.
3. Опублікувати сторінку на сервісі Github Pages.
